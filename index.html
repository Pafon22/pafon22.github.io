Carioca de nascimento, mineiro de coração.

Brasileiro nascido no Rio de Janeiro, RJ e criado a maior parte da vida em Volta Grande, Minas Gerais. Um pequeno munícipio de 5 mil habitantes.

Atualmente moro em Juiz de Fora, MG.

    Possuo competência em linguagens de programação como Java para aplicações e C# para criação de jogos na Unity3D, mas atualmente atuo usando:

  • Javascript para programação em geral;
  • Sony Vegas / Adobe Premiere / Anime Studio / Photoshop CS6 para pequenas animações, gifs ou emotes;
  • StreamElements / Nightbot / Mix It Up / StreamLabs / OBS para consumo de APIs, websockets, criação de chatbots e configurações auxiliares para produtores de conteúdo, principalmente na Twitch.
